# Python Fundamentals Tasks 🚀

A GitHub repository containing clean, well-structured solutions for three practical Python assignments. These tasks focus on solidifying core programming fundamentals, including variables, data type casting, logical operators, and deeply nested conditions.

---

## 📁 Project Structure

The folder contains 3 executable Python scripts:

### 1. Instant Discount Calculator (`Task_1.py`)
* **Objective:** Mastering basic conditional paths, handling user inputs, and performing data type casting (`float`).
* **How it works:** Prompts the user for their total purchase amount, calculates a tiered discount dynamically based on spending thresholds, and outputs both the exact discount value and the final total price.

### 2. Job Application Eligibility Portal (`Task_2.py`)
* **Objective:** Utilizing logical operators (`and`, `or`) to evaluate complex, multi-layered criteria while handling text input reliably.
* **How it works:** Filters applicants for a "Junior Python Developer" role based on strict corporate conditions: knowing Python is an absolute prerequisite, combined with either having 2+ years of experience/projects **OR** holding a CS degree/Bootcamp certification.

### 3. Smart ATM Simulator (`Task_3.py`)
* **Objective:** An intensive, deep-dive practice on heavily nested conditional statements (`if` / `elif` / `else`).
* **How it works:** Simulates a real-world ATM experience. It starts by validating a predefined 4-digit card security PIN. Upon successful verification, it presents a functional transaction menu to either (1) Cash Withdrawal or (2) Check Balance. For withdrawals, it strictly verifies funds availability before displaying the remaining account balance.

## 🛠️ Concepts & Tech Stack Used

* **Input Handling & Casting:** `input()`, `float()`, `int()`
* **String Sanitization:** `.lower()` and `.strip()` to ensure input checks are immune to accidental spaces or capitalization errors.
* **Comparison & Logical Operators:** `==`, `>=`, `<`, `and`, `or`
* **Control Flow:** Structured `if`, `elif`, `else` conditions alongside deeply `nested if` scopes.
* **Output Formatting:** Native Python `f-strings` for clean, dynamic console logging.
